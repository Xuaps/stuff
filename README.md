# stuff — Things Web

A local-first web clone of Things 3. Vanilla ES modules, no build step, IndexedDB persistence.

**Live: https://xuaps.github.io/stuff/** — deployed via GitHub Pages (deploy from branch `main`, root). Every push to `main` goes live automatically; no CI, no build.

## Develop

- `npm test` — run store tests (node:test)
- Open `index.html` locally or serve the folder with any static server
